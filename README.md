OJT Project with Laravel using NUXTJS/Axios 
#Set Up
git clone this repo link in your folder
1.-git clone <> 

Set up for Backend laravel 
1. cd api 
2. composer install
3. php artisan serve
check .env file  in api folder

Setup for frontend NUXTJS
1. cd client
2. npm install or npm install --force
3. npm run dev
